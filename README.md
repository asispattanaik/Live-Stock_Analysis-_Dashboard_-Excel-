# Live-Stock_Analysis-_Dashboard_-Excel-
I built an interactive Excel dashboard that tracks and analyzes a stock investment portfolio in real time. The system allows users to input stock details such as symbol, sector, market cap, number of shares, price paid, and current price.


Dashboard Image: https://github.com/asispattanaik/Live-Stock_Analysis-_Dashboard_-Excel-/blob/main/Screenshot%202025-11-24%20020543.png?raw=true
Data Set:https://docs.google.com/spreadsheets/d/1LYCBqrYyhdLCDl12qu8G1IPXAgVAugt2Tm3b8kmtNa4/edit?usp=sharing

**Conclusion**
This project successfully delivers a complete, easy-to-use Excel dashboard that helps investors track and understand their stock portfolio in real time. By combining automated calculations with clear visualizations, the dashboard provides meaningful insights into gains, losses, diversification, and risk exposure. It not only simplifies portfolio monitoring but also supports smarter decision-making by presenting complex financial data in a structured and intuitive format.
